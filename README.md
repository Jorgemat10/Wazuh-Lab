<h1>SIEM Monitoring Lab - Wazuh</h1>

<h2>Description</h2>
Wazuh is an open-source Security Information and Event Management (SIEM) solution that provides log analysis, file integrity checking, intrusion detection, vulnerability detection, and more.

In this project, Wazuh is used to monitor Windows and Linux machines, detect threats, and visualize activity via its web dashboard. This setup enhances network visibility and enables proactive defense strategies.

<h2>Tools and Technologies Used</h2>
- <b>Wazuh Manager</b>
- <b>Wazuh Web Interface</b>
- <b>Filebeat</b>
- <b>Elastic Stack</b>

<h2>Environments Used</h2>
- <b>Ubuntu Server (Wazuh Manager)</b>
- <b>Wazuh Web UI</b>
- <b>Monitored Clients: Windows 10 & Linux VMs</b>
